## BertNet: Harvesting Knowledge Graphs from PLMs

![](framework.png)

### Environment
We use `python 3.8` and all the required packages can be installed by pip:
```
pip install -r requirements.txt
```

### Start server
```
FLASK_ENV=development python server_main.py
```
